# Flask JWT Authentication System

A complete authentication system built with Flask using JWT (JSON Web Token) authentication.  
This project includes:

- User Signup
- User Login
- Access Token
- Refresh Token
- Protected Profile Route
- Refresh Token Route
- Logout System (optional if implemented)

Database used: SQLite3

---

# Tech Stack

- Python
- Flask
- Flask-JWT-Extended
- SQLite3
- Werkzeug Security
